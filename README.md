# 2D-Roguelike
A 2D bullet-hell roguelike game set in a medieval fantasy world. 

Stable is the 'Dev' Branch where we will post patches. Main will be the final product.



**__Setting up to be able to edit__**

Downloading Godot:
***Get the .net Version***
Run Godot Stable, not the one with console
Note:
For Godot you most likely will have to install the .net architecture, get it from get.dot.net & download 10.0

Getting Godot working:
Download Vs Code (Blue Icon)
Clone Repo on github desktop (or however you do it) and open with VS Code
Should prompt to download required extensions(If not its just 'Godot Tools' in extensions tab)

Need to select the godot executable and restart ***Make sure the folder 'Godot Sharp' is in the same location***
-- I have mine in C:/ProgramFiles/Godot-DotNet (I dont think location really matters)
Reload VS Code
Once connecting is done in the bottom right you should be able to open the editor
